
'error_T_S.m': Temperature and salinity diagrams for BTS's reports using cruise direct data.

'graficas_lineas_ext.m': Graphics of sampling lanes and hydrographic variables from BTS* cruise.

'graficas_lineas_int.m': Sampling lanes graphic with geographical coordinates data from BTS* cruise.

'lectura_files_CTD.m': Read 'up'/'down' CTD files from cruises. Allow to verify if is there a 'bad' cast. 

'geo_res_station.m': Allow to get the geographical coordinates of the stations when the CTD is in the sea bottom for every cast. Reshape up/down arrays. For report table generation purposes.

'geo_coord_station.m': Allow to get the geographical coordinates of the stations when the CTD is in the sea bottom for every cast. For report table generation purposes. Generate the calibrated cruise data
